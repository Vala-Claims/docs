## Overview

Release notes will summarize meaningful changes to Vala and link to updated guidance.

## When to use

Check this page when you notice a changed workflow or want to learn what is new.

## Prerequisites

No prerequisites are required.

## Step-by-step

<Note>
Release history will be added here as customer-facing updates are approved for publication.
</Note>

1. Review the newest entry first.
2. Open any linked Help Center pages for updated instructions.
3. Share workflow changes with affected team members.

## Best practices

- Review changes before using them in active matters.
- Confirm whether an update affects your organization's configured workflow.

## Common mistakes

- Assuming every feature is available to every role or organization
- Continuing to use an outdated workflow without checking updated guidance
