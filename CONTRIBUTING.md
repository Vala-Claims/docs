# Contributing to the Vala Help Center

The Help Center is a customer-facing extension of the Vala application. Keep changes clear, accurate, and aligned with the product.

## Writing guidelines

- Use active voice and address the reader as “you.”
- Keep paragraphs short and instructions scannable.
- Use sentence case for headings.
- Bold interface labels only when the label is confirmed in the product.
- Use placeholders when behavior has not been verified.
- Never invent product behavior, permissions, limits, or availability.
- Remind readers to review AI-generated information against sources when the workflow is consequential.

## Page structure

Customer-facing pages use these sections in order:

1. Overview
2. When to use
3. Prerequisites
4. Step-by-step
5. Best practices
6. Common mistakes
7. Related pages

## Before submitting

1. Preview the site with `mint dev`.
2. Run `mint validate`.
3. Run `mint broken-links`.
4. Check light and dark mode.
5. Verify navigation, links, and product terminology.
6. Confirm that the dependency manifest still points to valid pages.
